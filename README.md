> Javascript Client for the SUSI application Server.


## Install

```sh
$ npm install --save susi-client-js
```


## Usage

```js
var susiClientJs = require('susi-client-js');

susiClientJs('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
